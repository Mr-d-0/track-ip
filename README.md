# track-ip

## Installation :

* `apt update`
* `apt install git curl -y`
* `git clone https://github.com/Mr-d-0/track-ip.git`
* `cd track-ip`

#### > Run : `bash trackip`

## Single Command :
```
apt update ; apt install git curl -y ; git clone https://github.com/Mr-d-0/track-ip.git ; cd track-ip ; bash trackip
```
